# 코인선물 상하방 전략 체크봇 (TradingCheckBot)

바이낸스 **USDT-M 선물** 시세를 받아 여러 보조지표와 전략을 종합해
현재 시점의 **상방(LONG) / 하방(SHORT) / 중립(관망)** 신호를 보여주는 Windows 데스크톱 프로그램입니다.

- 타임프레임 선택: **1분 · 5분 · 15분 · 1시간 · 4시간 · 일봉 · 주봉 · 월봉**
- 심볼 입력(기본 BTCUSDT), 자동 갱신(15초)
- 캔들차트 + EMA(20/50/200) 오버레이 직접 렌더링
- API 키 불필요 (공개 시세 엔드포인트만 사용)

> ⚠️ 본 프로그램은 보조지표 기반 참고용 신호이며 투자 권유가 아닙니다. 투자 책임은 본인에게 있습니다.

---

## 사용 보조지표 & 전략

각 지표가 **상방(+1) / 중립(0) / 하방(-1)** 으로 투표하고, 합산 순점수로 종합 판정합니다.

| 지표 | 판정 기준 |
|------|-----------|
| 가격 vs EMA200 | 위=상방(장기 상승추세) / 아래=하방 |
| EMA20 vs EMA50 | 정배열=상방 / 역배열=하방 |
| EMA50 vs EMA200 | 골든 영역=상방 / 데드 영역=하방 |
| RSI(14) | ≥55 상방, ≤45 하방, ≥70 과매수(하방주의), ≤30 과매도(상방가능) |
| MACD(12,26,9) | 라인>시그널=상방 / 반대=하방 (히스토그램 부호 보조) |
| 볼린저밴드(20,2) | 중심선 위=상방 / 아래=하방, 밴드 이탈 시 반전 신호 |
| 스토캐스틱(14,3,3) | %K>%D=상방, 80↑ 과매수 / 20↓ 과매도 |
| ADX(14) | 추세 강도 표시(방향 투표엔 미포함) |

**종합 판정**: 순점수(상방표−하방표) ≥ +2 → 상방, ≤ −2 → 하방, 그 사이 → 중립.

## 🔎 단타 상승 시그널 스캐너 (신규)

메인 화면 상단의 **🔎 단타 스캐너** 버튼으로 실행합니다. 거래대금 상위 코인을
**1·3·5·15분봉**으로 동시에 스캔해 **상승 진입 점수(0~100)** 가 높은 순으로 랭킹합니다.
단타에 맞춰 빠른 설정(EMA 9/21/50)을 사용하며, 아래 **11가지 방향의 상승 시그널**을
가중 점수로 합산합니다.

| # | 상승 시그널(방향) | 핵심 판정 |
|---|------------------|-----------|
| 1 | EMA 정배열 | EMA 9>21>50 완전/부분 정배열 |
| 2 | EMA9 모멘텀 | 가격 > EMA9 + EMA9 상승중 |
| 3 | 거래량 폭발 | 직전 20봉 평균 대비 거래량 급증 |
| 4 | RSI 전환 | 과매도 반등 / 상승 모멘텀 회복 |
| 5 | MACD 크로스 | 히스토그램 음→양 골든크로스·확대 |
| 6 | 볼린저 돌파 | 중심선 상향 돌파 + 밴드 확장 |
| 7 | 스토캐스틱 | 과매도 상향 교차(%K>%D) |
| 8 | 고점 돌파 | 직전 10봉 고점 브레이크아웃 |
| 9 | 눌림목 반등 | EMA21 지지 후 양봉 반등 |
| 10 | 강세 캔들 | 장대양봉 · 연속 양봉 모멘텀 |
| 11 | ADX 추세 | 상승 방향 + 추세 강화 |

- **등급**: 75↑ 강한 매수 · 60↑ 매수 우위 · 45↑ 관심 · 그 외 관망
- 각 종목에 **ATR 기반 진입가 / 목표가 / 손절가(손익비 1:1.5)** 와 적중 근거를 표시
- 옵션: 스캔 종목 수(20~80), 최소 점수 필터, 30초 자동 스캔
- 공개 시세 엔드포인트만 사용(API 키 불필요), 동시 요청 8개로 제한

> ⚠️ 점수가 높다고 매수 신호가 아니며 변동성·뉴스·호가 흐름을 함께 확인하세요. 투자 책임은 본인.

## 빌드 / 실행

### 1. 개발 중 바로 실행
```powershell
dotnet run --project src\TradingCheckBot\TradingCheckBot.csproj
```

### 2. 단일 실행 파일(.exe) 만들기 — .NET 설치 불필요
```powershell
.\publish.ps1
# 결과: publish\TradingCheckBot.exe
```

### 3. 설치 프로그램(Setup.exe) 만들기
[Inno Setup](https://jrsoftware.org/isdl.php) 설치 후:
```powershell
.\build-installer.ps1
# 결과: installer\Output\CoinFF-TradingCheckBot-Setup.exe
```

---

## 요구 사항
- 빌드: .NET 9 SDK (Windows)
- 실행: Windows 10/11 x64 (단일 exe는 런타임 포함이라 별도 설치 불필요)
- 인터넷 연결 (바이낸스 `fapi.binance.com` 접속). 지역에 따라 차단될 수 있습니다.

## 프로젝트 구조
```
TradingCheckBot/
├─ src/TradingCheckBot/
│  ├─ Models/Candle.cs            캔들 데이터 모델
│  ├─ Services/BinanceClient.cs   바이낸스 선물 klines API
│  ├─ Indicators/Indicators.cs    EMA/RSI/MACD/볼린저/스토캐스틱/ADX 계산
│  ├─ Indicators/StrategyEngine.cs 지표 종합 → 상하방 점수화
│  ├─ MainWindow.xaml(.cs)        UI + 캔들차트 렌더링
│  └─ App.xaml(.cs)               앱 진입점/테마
├─ installer/setup.iss            Inno Setup 설치 스크립트
├─ publish.ps1                    단일 exe 게시
└─ build-installer.ps1            게시+설치 프로그램 일괄 빌드
```
